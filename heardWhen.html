Heard When

Web, Git, Vue

Description: A multiplayer music trivia game! Created with JavaScript, TypeScript, CSS, HTML, Vue, Firebase Link:

Background

HeardWhen is a music trivia game inspired by the Swedish board game “När då då?”, where players guess the chronological order of songs from different decades.

The objective is to collect as many songs or cards as possible, with the difficulty increasing as more are collected.

Game

Songs are sourced from Spotify’s decade-specific playlists via the Spotify Web API.

Players arrange songs on a shared board, placing them in the correct order based on release year.

If a guess is correct, the song is placed and the player can choose to Lock their progress or Continue for more points.

An incorrect guess discards that round’s cards and lets other players guess the exact year.

Technologies:

Built with JavaScript, TypeScript, CSS, HTML, Vue, Firebase.

The application features a setup menu, the main game board, a lock/continue option, and a year-guess input.

Game data includes player states, locked cards, guesses, and scores.

API:

Spotify Web API to fetch song title, artist, and year information

Challenges & Learnings

  • Learned how to effectively apply CSS for styling components and creating a clean, user-friendly design.
  • Improved skills in responsive design and layout techniques to ensure usability across different devices.
